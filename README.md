# EDC Arborismo - Landing Page & Digital Identity

Este proyecto comprende el desarrollo de una identidad digital integral y una landing page de alto impacto para **EDC Arborismo**, especialistas en arboricultura avanzada y poda de altura con certificaciones internacionales[cite: 1, 2].

## 🎯 El Concepto
Bajo la premisa **"Diseñar es decidir. Lo demás sobra"**, el objetivo fue proyectar una imagen de seguridad, precisión científica y profesionalismo extremo. Se buscó superar las expectativas del rubro mediante una interfaz técnica que comunica autoridad en la gestión de arbolado[cite: 1].

## 🛠️ Tecnologías Utilizadas
* **React**: Para una navegación fluida y arquitectura escalable[cite: 1].
* **Tailwind CSS**: Estilado de alta fidelidad con enfoque en "Dark Mode" y acentos vibrantes[cite: 1, 2].
* **Framer Motion**: Implementación de micro-interacciones y transiciones de página dinámicas[cite: 1].
* **Vite**: Entorno de desarrollo ágil para optimización de tiempos de carga[cite: 1].

## 🚀 Características Principales
* **Navegación Arquitectónica**: Separación clara entre Landing Page y Portfolio de proyectos[cite: 1].
* **Estructura de Servicios**: Segmentación estratégica en Consultoría, Intervenciones e Ingeniería de Sustentación[cite: 1].
* **Validación Técnica**: Sección dedicada a certificaciones ISA (International Society of Arboriculture) y estándares ANSI Z133[cite: 1, 2].
* **Portfolio Dinámico**: Galería de intervenciones técnicas con detalles específicos de proyectos (ej. Poda en Altos del Casco)[cite: 2].
* **Conversión Optimizada**: Integración de CTAs directos para presupuestos mediante WhatsApp y formularios personalizados[cite: 2].

## 🌲 Sobre el Cliente
EDC Arborismo no solo interviene árboles; aplica criterios científicos para garantizar su longevidad y la seguridad del entorno, operando bajo los estándares más altos de Argentina y el exterior[cite: 1, 2].
